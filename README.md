# ScalingClever
MonoGame 适配多种设备分辨率运行库。
功能：1.根据设备分辨率等比例放大游戏界面，全屏适配。2.触控点根据设备分辨率等比例放大，全屏适配
安装要求：monogame 3.5.1
使用方法：http://www.xnadevelop.com/ios/monogame-suitable-for-various-equipment-resolution-runtime-library-scalingclever/
